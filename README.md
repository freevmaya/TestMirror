Сцена - Assets\Vmaya\VNet\Scenes\Test.unity


Приложение может выступать как в роли сервера так и в роли сервера, в UI приложения есть соответствующие кнопки:

1. Client - подключится в режиме клиента.
2. Host - подключится в режиме сервера.
3. Disconnect - закрыть соединение/остановить сервер.
4. Close - закрыть программу.

# Примечение

1. После соединения с сервером, клиент отправляет запрос на подписку HelloMessage.
2. Сервер получив запрос на подписку HelloMessage, регистрирует подписку и сразу отправляет соответствующее сообщение клиенту.
3. Клиент получив сообщение HelloMessage выводит его текст в консоль приложения.

Подписка осущуствляется методом: Subscribe
Отправка подписчикам, методом: SendToSubscribers
